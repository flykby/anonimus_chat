# 008. Main menu

**Статус:** todo  
**Фаза:** bot  
**Зависимости:** 007, 009

## Описание

Главное меню после регистрации: три кнопки — Начать разговор, Профиль, Правила. Точка входа для всех основных сценариев.

## Scope

- Reply keyboard или inline keyboard с 3 пунктами:
  1. Начать разговор
  2. Профиль
  3. Правила
- Показ меню после успешной регистрации и при возврате из подразделов
- Блокировка меню во время активного диалога (кроме «Завершить диалог»)
- Handler маршрутизации: `menu:start_chat`, `menu:profile`, `menu:rules`
- API check: есть ли активный dialog у user

## Acceptance criteria

- [ ] Зарегистрированный пользователь видит меню на своём языке
- [ ] Каждая кнопка ведёт в соответствующий flow
- [ ] Во время диалога главное меню скрыто, видна только «Завершить диалог»
- [ ] Незарегистрированный пользователь при любом действии → регистрация

## Технические заметки

- Reply keyboard удобнее для «Начать разговор» (всегда на виду)
- `resize_keyboard=True`, `one_time_keyboard=False`
- Тексты кнопок из i18n (задача 009)
- Back navigation: из Профиля/Правил — кнопка «Назад» → меню

## Out of scope

- Реализация подразделов (отдельные задачи 010+, 023+, 027)
