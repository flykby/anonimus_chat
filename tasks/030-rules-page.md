# 030. Rules page

**Статус:** todo  
**Фаза:** profile  
**Зависимости:** 011, 009

## Описание

Страница с правилами использования бота на RU и EN. Доступна из главного меню.

## Scope

- Тексты правил в `bot/locales/rules_ru.md` / `rules_en.md` или yaml long-text
- Разделы:
  - Возраст 18+
  - Анонимность и запрет передачи личных данных
  - Поведение в P2P (запрет оскорблений, illegal content)
  - AI-персоны — развлекательный контент
  - Adult-контент и blur/premium
  - Платежи Stars, политика возвратов
  - Удаление аккаунта
  - Disclaimer: бот не гарантирует реальных людей в AI-режиме
- Кнопка «Назад» → главное меню
- Telegram: разбить на 2–3 сообщения если > 4096 символов

## Acceptance criteria

- [ ] Правила открываются из меню на языке пользователя
- [ ] RU и EN версии полные и согласованные
- [ ] «Назад» работает
- [ ] Текст покрывает premium, adult, P2P moderation

## Технические заметки

- Юридически нейтральный шаблонный текст, review перед launch (035)
- Версионирование rules: `rules_version` в event при регистрации optional
- HTML parse_mode для форматирования (bold заголовки)

## Out of scope

- Юридическая экспертиза
- Multi-page rules с pagination UI
