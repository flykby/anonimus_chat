# 038. AI dialog echo stub

**Статус:** done  
**Фаза:** bot  
**Зависимости:** 015  
**Milestone:** 2 — полный бот без real AI

## Описание

Пока нет RunPod/LLM (016+), AI-диалог (M→F, F→F) работает как **echo-бот**: пользователь пишет текст → бот повторяет. Завершение диалога через «Завершить диалог» (015).

## Scope

- В активном AI-диалоге текстовые сообщения → echo ответ
- Кнопка «Завершить диалог» работает как в 015
- P2P-диалоги **не** echo — relay в задаче 025

## Acceptance criteria

- [x] AI dialog: сообщение пользователя → тот же текст в ответ
- [x] End dialog flow не сломан
- [x] P2P dialog (когда появится) не попадает в echo

## Out of scope

- LLM, persona, контекст (016, 017, 036)
- Сохранение сообщений в `dialog_messages` (можно добавить позже)
